# 📖 Repository Purpose

Welcome to the **Cyber Threat Intelligence & Analysis Engineering** repository.

This repository is dedicated to building practical skills in **Cyber Threat Intelligence (CTI)** through hands-on labs, real-world threat investigations, malware intelligence, Indicator of Compromise (IOC) analysis, adversary profiling, and intelligence automation.

The objective is to provide security professionals, SOC analysts, incident responders, threat hunters, and blue teams with practical knowledge to collect, analyze, enrich, and operationalize cyber threat intelligence for proactive defense.

---

# 🎯 Objectives

✅ Learn Cyber Threat Intelligence lifecycle

✅ Collect threat intelligence from multiple sources

✅ Analyze malware campaigns

✅ Extract Indicators of Compromise (IOCs)

✅ Track Advanced Persistent Threats (APTs)

✅ Investigate phishing infrastructure

✅ Analyze attacker Tactics, Techniques and Procedures (TTPs)

✅ Map threats to MITRE ATT&CK

✅ Build actionable intelligence reports

✅ Automate threat intelligence workflows
